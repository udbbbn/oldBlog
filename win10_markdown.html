解决win10打不开markdown报错'Awesomium.Windows.Controls.WebControl'

Date 周六 29 七月 2017 Tags markdown / win10

问题描述

将开发环境换到笔记本后发现markdown打不开 报错'Awesomium.Windows.Controls.WebControl'，如下图：

解决方案

命令行输入regedit 将下面的值改为0 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Lsa\FipsAlgorithmPolicy\Enabled

再次启动markdown 若会报html渲染错误 安装awwsomium 这个软件后重启即可。
